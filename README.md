# ESPHomeNow
Use ESPNow integrated directly with ESPHome and HomeAssistant

<img width="1032" height="604" alt="image" src="https://github.com/user-attachments/assets/f9ee1149-8669-4921-aa2b-0ab80c10caae" />

Provides for 
* MQTT discovery of entities
* Queued commands for offline or sleeping nodes
* OTA over WiFi

Entities Implemented:
* Switch
* Button
* Sensor
* Text
